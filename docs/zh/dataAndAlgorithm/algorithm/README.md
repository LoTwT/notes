## 复杂度

算法的时间复杂度是一个函数，通常用大 O 表示，定性描述该算法的运行时间，一般忽略首项系数和低阶项。

算法的空间复杂度描述该算法执行需要的内存空间。

但是现在存储价格实在是太便宜啦...所以一般只考虑时间复杂度 (用空间换时间)...
